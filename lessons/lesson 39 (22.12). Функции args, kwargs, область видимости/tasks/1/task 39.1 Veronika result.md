### 6. Вариант полного решения
```python
def mean_abs_deviation(*scores):
    if not scores:
        raise ValueError("Необходимо передать хотя бы один балл")
    mean = sum(scores) / len(scores)
    deviations = [round(abs(score - mean), 1) for score in scores]
    return tuple(deviations)
```

**Тесты:**
```python
print(mean_abs_deviation(3, 5, 7, 9))  # (3.0, 1.0, 1.0, 3.0)
print(mean_abs_deviation(*[10, 8, 6]))  # (2.0, 0.0, 2.0)
```
