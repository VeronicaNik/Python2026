# Задание 33.4. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Set comprehensions» показывает. как коротко собрать множество уникальных значений после простой проверки.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Есть список `raw_cities`, где встречаются пустые строки и повторяющиеся названия. Построй множество `clean_cities`, содержащее только непустые города в нижнем регистре. Используй одно выражение set comprehension.

## Пример входных данных
```
raw_cities = ["Москва", "", "москва", "Пермь", " Казань "]
```

## Пример выходных данных
```
{'москва', 'пермь', 'казань'}
```

## Критерии успешного выполнения
- Множество `clean_cities` формируется единственным set comprehension.
- Пустые строки и строки с пробелами после `strip()` игнорируются.
- Все города приводятся к нижнему регистру.
- Исходный список `raw_cities` не меняется.

## Подсказки
- Сначала убери пробелы по краям через `strip()` и сохрани результат во временной переменной.
- Проверяй непустоту строки перед добавлением в множество.
- Используй `lower()` только после того, как убедишься, что строка не пустая.

## Частые ошибки и ограничения
- Не оборачивай comprehension в `set()` и не используй `add`.
- Не забывай про пустые строки: они не должны попадать в итог.
- В списках не превышай пять пунктов.
