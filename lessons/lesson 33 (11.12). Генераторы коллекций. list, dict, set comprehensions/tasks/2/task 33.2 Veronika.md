# Задание 33.2. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«List comprehensions» показывает. как одним коротким выражением заменить цикл и сразу получить готовый список.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Есть список `orders`, где каждая запись — кортеж `(номер, статус, сумма)`. Построй новый список строк `ready_orders`, куда попадут только заказы со статусом `"готов"` и суммой не меньше 500. Формат строки: `"Заказ #<номер>: <сумма> ₽"`. Используй одно выражение list comprehension.

## Пример входных данных
```
orders = [
    (101, "в работе", 450),
    (102, "готов", 510),
    (103, "готов", 700),
    (104, "отменён", 900)
]
```

## Пример выходных данных
```
['Заказ #102: 510 ₽', 'Заказ #103: 700 ₽']
```

## Критерии успешного выполнения
- Список `ready_orders` создаётся одним list comprehension.
- В итог попадают только заказы со статусом `"готов"` и суммой ≥ 500.
- Формат строки строго как в примере.
- Исходный список `orders` остаётся без изменений.

## Подсказки
- Вспомни шаблон `[результат for элемент in список if условие]`.
- Распакуй значения прямо в выражении: `for number, status, total in orders`.
- Сравни статус строкой `== "готов"` и сумму числом `>= 500`.
- Строку удобно собрать через f-строку `f"Заказ #{number}: {total} ₽"`.
- Сохрани новый список в переменной `ready_orders` и распечатай его для проверки.

## Частые ошибки и ограничения
- Не вставляй дополнительных циклов или `append`.
- Не забывай про оба условия фильтрации.
- Следи за тем. чтобы в списках было не больше пяти пунктов.
