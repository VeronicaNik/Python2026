Ключом в новом словаре должно быть название курса, а значением — строка `"пройден"`, если `score >= 70`, иначе `"повторить"`. Решение должно быть реализовано одним dict comprehension без изменения исходного списка.

     ```python
                      if cours["score"]>=70}
     ```
