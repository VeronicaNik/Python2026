# Задание 33.3. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Dict comprehensions» помогает быстро собрать новый словарь из списка данных и сразу определить нужные подписи.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Есть список `courses`, где каждый элемент — словарь с ключами `"title"` и `"score"`. Построй словарь `course_status`, в котором ключ — название курса, а значение — строка `"пройден"`, если балл ≥ 70, иначе `"повторить"`. Используй одно выражение dict comprehension и не меняй исходный список.

## Пример входных данных
```
courses = [
    {"title": "Основы Python", "score": 80},
    {"title": "Git", "score": 65},
    {"title": "SQL", "score": 72}
]
```

## Пример выходных данных
```
{'Основы Python': 'пройден', 'Git': 'повторить', 'SQL': 'пройден'}
```

## Критерии успешного выполнения
- Весь словарь `course_status` строится одним dict comprehension.
- Условие `score >= 70` определяет текст для значения.
- Исходный список не изменяется и не копируется вручную.
- Ключи словаря совпадают с названиями входных курсов.

## Подсказки
- Пройдись по списку словарей как `for course in courses`.
- Выбор строки сделай через тернарный оператор.
- Сохрани результат в новой переменной и выведи для проверки.

## Частые ошибки и ограничения
- Не создавай цикл `for` с `dict()` вокруг comprehension.
- Не забывай про условие на балл.
- Следи, чтобы списки имели не более пяти пунктов.
