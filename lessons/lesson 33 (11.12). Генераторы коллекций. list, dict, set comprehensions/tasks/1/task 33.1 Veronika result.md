### 6. Полное решение
```python
animals = ["кот", "бегемот", "лось", "рысь", "жираф", "пёс"]
short_names = [animal.upper() for animal in animals if len(animal) < 5]
```

**Объяснение**:
- `animal.upper()` переводит слово в верхний регистр.
- `if len(animal) < 5` фильтрует слова короче 5 символов.
- Результат сохраняется в `short_names`.
