# Задание 33.1. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«Введение в генераторы коллекций» — учимся записывать простое преобразование списка в одно понятное выражение без ручного цикла.

## Условие задачи
**Сложность**: <span style="color: #2e7d32">● лёгкий</span>. примерное время: 15–20 минут

Есть список названий животных. Нужно получить новый список `short_names`, где остаются только слова короче 5 символов, а каждое записано заглавными буквами. Используй один list comprehension.

## Пример входных данных
```
animals = ["кот", "бегемот", "лось", "рысь", "жираф", "пёс"]
```

## Пример выходных данных
```
['КОТ', 'ЛОСЬ', 'РЫСЬ', 'ПЁС']
```

## Критерии успешного выполнения
- Используешь ровно одно выражение list comprehension.
- В результат попадают только слова длиной < 5 символов.
- Каждое слово переведено в верхний регистр методом `upper()`.
- Исходный список `animals` не меняется.

## Подсказки
- Вспомни форму `[выражение for слово in animals if условие]`.
- Чтобы убрать длинные слова. добавь условие `if len(слово) < 5`.
- Для перевода в верхний регистр используй `слово.upper()`.
- Сохраняй новый список в переменную `short_names`.

## Частые ошибки и ограничения
- Не ставь `print` внутри comprehension — результат должен быть списком.
- Не забывай про условие фильтрации, иначе в список попадут все слова.
- Избегай сложных конструкций. задача должна читаться с первого взгляда.
