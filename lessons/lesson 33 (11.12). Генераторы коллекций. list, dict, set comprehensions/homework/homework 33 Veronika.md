# Домашнее задание. Урок 33. Генераторы коллекций. list, dict, set comprehensions

## Краткая сводка
- Проблемные блоки: list comprehension с фильтрацией и форматированием, dict comprehension с условными значениями, set comprehension с очисткой строк.
- Сильные стороны: аккуратная фильтрация и форматирование заказов (task 33.2 на 100 баллов).
- Приоритеты: закрепить преобразование данных в едином выражении, научиться выводить строковые статусы вместо чисел, отработать нормализацию строк в множествах.

## Задача 1. Повторение сложной темы. «Короткие названия животных»
- **Блок:** list comprehension.
- **Описание:** дан список `animals` со строками разной длины. Собери список `short_names` только из названий длиной < 5 символов, переведённых в верхний регистр, сохранив порядок.
- **Пример входа:** `["кот", "бегемот", "лось", "рысь", "жираф", "пёс"]`.
- **Критерии:** один list comprehension, формат элементов `"ЖИРАФ"` без разбиения на символы, исходный список не меняется, никаких `print` внутри comprehension.

## Задача 2. Закрепление практики. «Отгрузки готовых заказов 2.0»
- **Блок:** list comprehension с форматированием (усилить сильную сторону).
- **Описание:** у тебя есть список `orders = [(101, "готов", 520), (102, "в работе", 450), ...]`. Сформируй список строк `"Заказ #ID: SUM ₽"` только для записей со статусом `"готов"` и суммой ≥ 500. Добавь проверку, что ID чётный, чтобы имитировать выборку по смене.
- **Критерии:** один list comprehension, правильный формат строки (пробел перед `₽`), исходный список не меняется, все условия (`status`, `sum`, чётность ID) в одном выражении.

## Задача 3. Интеграция блоков. «Статусы онлайн курсов»
- **Блок:** dict comprehension.
- **Описание:** список словарей `courses = [{"title": "Python", "score": 82}, {"title": "Git", "score": 65}, ...]`. Построй словарь `course_status`, где значение `"пройден"`, если `score ≥ 70`, иначе `"повторить"`. Все курсы должны попасть в словарь.
- **Критерии:** один dict comprehension, тернарный оператор внутри значения, никаких фильтров `if` после comprehension, исходный список не меняется.

## Задача 4. Подготовка к следующему уроку. «Чистый список городов»
- **Блок:** set comprehension и предобработка строк.
- **Описание:** дан список строк `raw_cities = ["Москва", "", " москва ", "Пермь", " Казань "]`. Собери множество `clean_cities` с городами в нижнем регистре, без пустых строк и лишних пробелов.
- **Критерии:** один set comprehension, используешь `strip()` и `lower()`, условие `if city.strip()` внутри comprehension, никакой попытки вызвать строковые методы у множества при выводе.
