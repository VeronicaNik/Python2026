6. **Полное решение**  
   ```python
   rides_by_driver = {}
   shift_log = [('drv-1', 6), ('drv-2', 4), ('drv-1', 5), ('drv-3', 2)]
   finished = ['drv-2']

   # Суммирование поездок
   for driver, rides in shift_log:
       rides_by_driver[driver] = rides_by_driver.get(driver, 0) + rides

   # Удаление водителей из finished
   for driver in finished:
       result = rides_by_driver.pop(driver, None)
       if result is None:
           print(f"Водитель {driver} не найден")

   # Вывод результата
   print(rides_by_driver)
   ```
