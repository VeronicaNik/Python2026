## Задача 1. Учёт бронирования столиков

Пример входных данных:
```python
bookings = [
    ("window", "18:00"),
    ("center", "19:30"),
    ("window", "21:00"),
]
check = ["window", "terrace"]
```

### Требуется
1. Собрать словарь `last_booking` по столикам.
2. Для каждого значения из списка `check` вывести `<место> -> <время или "свободен">`, используя `get`.

## Задача 2. Корректная мутация словаря

Пример входных данных:
```python
logs = [
    ("chat", 12),
    ("feed", 7),
    ("chat", 5),
]
remove = ["feed", "archive"]
```

### Требуется
1. Собрать словарь `minutes`, где ключ. раздел, значение. суммарное время.
2. Для каждого раздела из `remove` удалить запись через `pop(name, 0)`.
3. Печатать только финальный словарь.

## Задача 3. Множества жанров

Пример входных данных:
```python
a = {"fantasy", "romance", "detective"}
b = {"fantasy", "sci-fi"}
```

### Требуется
1. Найти множество жанров, которые есть и в `a`, и в `b`.
2. Найти множество жанров, которые есть только в `b`.
3. Вывести результат с подписями.

## Задача 4. Подготовка к comprehensions

Пример входных данных:
```python
articles = [
    ("python", 5),
    ("sql", 3),
    ("python", 4),
]
limit = 5
```

### Требуется
1. Собрать словарь `reads_by_topic`.
2. Получить новый словарь только для тем, где суммарные чтения не меньше `limit`.
3. Распечатать результат.
