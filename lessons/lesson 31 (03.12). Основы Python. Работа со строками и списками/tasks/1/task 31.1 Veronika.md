### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая нормализует строку с именем пользователя и выводит краткую статистику.

Требования:
- убрать пробелы по краям строки
- привести имя к виду `ПерваяБукваЗаглавная` остальные строчные
- вывести первую и последнюю букву имени
- вывести длину имени

### Пример входных данных
```
  вЕроНИка  
```

### Пример выходных данных
```
Имя: Вероника
Первая буква: В
Последняя буква: а
Длина: 8
```

### Критерии успешного выполнения
- строка очищена от лишних пробелов
- регистр имени приведен к нужному виду
- первая и последняя буквы определены корректно
- длина совпадает с количеством символов имени

### Подсказки
- используй `strip`, `title` или комбинацию методов
- для индексов используй `[0]` и `[-1]`

### Частые ошибки и ограничения
- учитывай, что строка может оказаться пустой после очистки
- не используй внешние библиотеки
