### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая помогает работать со списком фильмов для просмотра.

Требования:
- прочитать строку с фильмами, разделенными запятой
- удалить пробелы по краям каждого названия
- привести все названия к одному регистру (например. нижнему)
- удалить дубликаты и отсортировать список по алфавиту

### Пример входных данных
```
  Интерстеллар, матрица, матрица, Дюна,  интерстеллар 
```

### Пример выходных данных
```
['дюна', 'интерстеллар', 'матрица']
```

### Критерии успешного выполнения
- строка разбита на элементы по запятой
- пробелы и дубликаты убраны
- список отсортирован по алфавиту

### Подсказки
- используй `split(',')`, `strip()`, `lower()`
- дубликаты можно убрать через `set`, затем отсортировать

### Частые ошибки и ограничения
- не полагайся на порядок элементов после преобразования во множество
- не используй внешние библиотеки
