### Блок 1: Работа со списками

Ты готовишь меню для ресторана. Создай список основных блюд, добавь к нему десерты из списка `десерты`, выведи общее количество блюд, покажи первое и последнее блюдо, а затем выведи нумерованный список всего меню.

Пример входных данных:
```python
main_dishes = ["стейк", "паста", "салат"]
desserts = ["чизкейк", "мороженое", "тирамису"]
```

### Блок 2: Срезы

У тебя есть список заказов за день (номера заказов). 
1. Выведи первые четыре заказа.
2. Выведи последние три заказа.
3. Выведи каждый второй заказ, начиная с первого.
4. Выведи список без первого и последнего заказа.

Пример входных данных:
```python
orders = [101, 102, 103, 104, 105, 106, 107, 108, 109]
```

### Блок 3: Мутация списков

У тебя есть список ингредиентов для блюда. 
1. Добавь в конец списка ингредиенты: "соль" и "перец".
2. Вставь в середину списка (после второго элемента) ингредиент "чеснок".
3. Удали из списка ингредиент "лук" (если он есть).
4. Перемести последний ингредиент в начало списка.

Пример входных данных:
```python
ingredients = ["лук", "морковь", "картофель", "томаты"]
```

### Блок 4: Вложенные структуры

У тебя есть список блюд в меню, каждое блюдо представлено словарем с полями: `название`, `тип` (основное/десерт/напиток), `цена` (в рублях). 
1. Распечатай информацию в формате: `Название: тип (цена руб)`.
2. Сформируй список названий блюд с ценой больше 500 рублей.
3. Создай новый список только с десертами (тип "десерт").
4. Подсчитай среднюю цену блюда.

Пример входных данных:
```python
menu = [
    {"название": "стейк", "тип": "основное", "цена": 800},
    {"название": "паста", "тип": "основное", "цена": 450},
    {"название": "чизкейк", "тип": "десерт", "цена": 300},
    {"название": "тирамису", "тип": "десерт", "цена": 350}
]
```
