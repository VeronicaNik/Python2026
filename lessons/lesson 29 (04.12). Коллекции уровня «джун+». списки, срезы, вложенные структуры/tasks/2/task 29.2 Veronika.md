# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Практикуем получение разных подсписков и работу с шагом.

## Условие задачи
Ты ведёшь список расходов на подготовку к празднику. Нужно:
1. Получить первые пять трат, чтобы свериться с начальными покупками.
2. Показать три последние траты для контроля финального бюджета.
3. Сформировать срез, который включает элементы с индекса 2 по 6 включительно.
4. Собрать каждый второй элемент, начиная со второго индекса, чтобы проверить повторы.

## Пример входных данных
```python
costs = [1200, 800, 560, 900, 450, 700, 650, 400, 300]
```

## Пример выходных данных
```
Первые покупки: [1200, 800, 560, 900, 450]
Финальные траты: [700, 650, 400, 300]
Средний блок: [560, 900, 450, 700, 650]
Каждый второй со 2 индекса: [560, 450, 650, 300]
```

## Критерии успешного выполнения
- Все выборки построены срезами без циклов
- Используются разные комбинации start, stop и step
- Каждый результат подписан
- Исходный список не меняется

## Подсказки
- Хвост списка удобно брать `[-n:]`
- Чтобы включить элемент с индекса 6, укажи `stop=7`
- Шаг `2` позволит пропускать элементы
- Пустая граница экономит символы, когда нужен старт или конец списка

## Частые ошибки и ограничения
- Не ставь шаг 0
- Проверяй, чтобы границы не выходили за пределы списка
- Не перепутай порядок `start` и `stop`
- Не модифицируй исходный список, работай с копиями через срезы
