# Задание 29.4. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Вложенные структуры. Учимся работать с вложенными словарями и списками, чтобы аккуратно собирать отчеты из табличных данных.

## Условие задачи
Ты редактируешь школьный научный журнал:
1. Создай список словарей `articles` с ключами `section`, `title`, `authors` (список имен), `status`.
2. Выведи содержание в формате `Раздел: section. статья: title. авторы: ...`.
3. Собери множество всех авторов через обход вложенных списков `authors`.
4. Построй словарь `section -> количество статей`, подсчитав элементы внутри каждой категории.
5. Отфильтруй статьи со статусом `ready` и выведи их названия отдельным списком.

## Пример входных данных
```python
articles = [
    {"section": "биология", "title": "Рост растений", "authors": ["Ника", "Олег"], "status": "draft"},
    {"section": "физика", "title": "Световые эксперименты", "authors": ["Саша"], "status": "ready"},
    {"section": "биология", "title": "Экосистемы", "authors": ["Ника"], "status": "ready"}
]
```

## Пример выходных данных
```
Раздел: биология. статья: Рост растений. авторы: Ника, Олег
Раздел: физика. статья: Световые эксперименты. авторы: Саша
Раздел: биология. статья: Экосистемы. авторы: Ника
Все авторы. {'Ника', 'Олег', 'Саша'}
Статистика по разделам. {'биология': 2, 'физика': 1}
Готовые статьи. ['Световые эксперименты', 'Экосистемы']
```

## Критерии успешного выполнения
- Статьи представлены списком словарей с одинаковыми ключами
- Вывод содержания формируется циклом без хардкода индексов
- Уникальные авторы собираются автоматически
- Фильтр по статусу не меняет исходный список

## Подсказки
- Используй `", ".join(entry["authors"])`, чтобы вывести имена
- Уникальные авторы удобно хранить во множестве
- Для статистики по разделам примени `dict.setdefault(section, 0)`

## Частые ошибки и ограничения
- Не редактируй исходный список при фильтрации
- Следи, чтобы ключи словаря были написаны одинаково
- Не дублируй код для каждого раздела. цикл решит задачу
- Не забывай распечатать итоговые структуры после вычислений
