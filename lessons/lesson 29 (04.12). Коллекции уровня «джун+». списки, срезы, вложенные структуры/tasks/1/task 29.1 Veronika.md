# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Важно освоить создание списка, обращение к элементам по индексу и перебор без ошибок выхода за границы.

## Условие задачи
Тебе нужно организовать список дел для подготовки к семейному празднику. Создай стартовый список покупок, добавь идеи родственников, выведи общее количество пунктов, покажи элемент с индексом 2 и последний пункт. Заверши вывод пронумерованным списком, чтобы легко ставить галочки.

## Пример входных данных
```python
shopping = ["фрукты", "сыр", "свежий хлеб"]
family_bonus = ["свечи", "салфетки"]
```

## Пример выходных данных
```
Всего пунктов: 5
Элемент №2: сыр
Последний пункт: салфетки
1. фрукты
2. сыр
3. свежий хлеб
4. свечи
5. салфетки
```

## Критерии успешного выполнения
- Список создаётся в коде и расширяется методами списка
- Количество элементов считается через `len`
- Индексы используются для выбора внутреннего и последнего элементов
- Итоговый список выводится с нумерацией через цикл

## Подсказки
- Чтобы вставить элементы сразу, используй `extend`
- Индекс `2` обращается к третьему элементу, учти это при выводе
- Для нумерации подключи `enumerate(..., start=1)`

## Частые ошибки и ограничения
- Не оставляй список пустым перед обращением к индексу `2`
- Избегай ручного счёта элементов, доверяй `len`
- Не выводи список в одну строку через `print(list)`, разворачивай в цикл
- Проверяй, что новые пункты добавлены до финального вывода
